# Ore-Ore-WAF

A Web Application Framework Dedicated To Me.
===============



# INSTALLATION

Recommended environment : Plenv and Carton


```
 $ git clone https://github.com/adokoy001/Ore-Ore-WAF.git
 $ cd Ore-Ore-WAF
 $ carton install 

```

# RUN

```
 $ ./command develop
   or
 $ ./command production
```

# STOP

```
 $ ./command stop
```
